# MODICIA-O.S.
Linux-based operating system designed for performance and simplicity

**Modicia O.S.** is a multimedia-oriented Linux distribution designed for creative professionals: videomakers, musicians, graphic designers, photographers, and educators. Built on Debian Stable, it offers high performance, professional tools out of the box, and a sleek, intuitive interface.

## 🎯 Project Goals

- Deliver a complete, ready-to-use operating system
- Optimize performance for modern hardware
- Empower creativity with top-tier open-source software

## 🚀 Key Features

- Optimized Linux kernel (Real-Time available)
- Desktop Environment: Customized Cinnamon
- Over 2,000 preinstalled applications for multimedia and productivity
- Compatibility with both modern and legacy hardware
- Live USB mode for portable use without installation

## 📦 Installation

1. Download the ISO image from the [official website](https://www.modiciaos.cloud/)
2. Burn it to a DVD or create a bootable USB stick
3. Boot into live mode or install it on your system

## ❓ Frequently Asked Questions

**Is Modicia O.S. free?**  
Yes, the community edition is completely free to download and use.

**What is the system base?**  
Modicia O.S. is based on Debian Stable with a customized Cinnamon desktop.

**Is it beginner-friendly?**  
Absolutely! The interface is intuitive and the system is ready to use even for Linux newcomers.

**Can I install additional software?**  
Yes, via the integrated App Center with over 1,000 `.deb` and `.AppImage` packages.

**Does it support Real-Time Kernel?**  
Yes, a one-click utility is included to install and configure it.
