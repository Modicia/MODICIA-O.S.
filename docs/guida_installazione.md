# Installation Guide — Modicia O.S.

Welcome to Modicia O.S., a powerful and elegant Linux distribution designed for multimedia professionals and creative users. This guide will help you install Modicia O.S. on your computer in just a few steps.

---

## 🖥️ System Requirements

- **Processor**: 64-bit Intel or AMD
- **RAM**: Minimum 2 GB (4 GB or more recommended)
- **Storage**: At least 20 GB of free disk space
- **USB/DVD**: For bootable media

---

## 📥 Step 1: Download the ISO

Visit the [official website](https://www.modiciaos.cloud/) and download the latest Modicia O.S. ISO image.

---

## 🔧 Step 2: Create Bootable Media

Use one of the following tools to create a bootable USB stick:

- **Windows**: [Rufus](https://rufus.ie/)
- **Linux/macOS**: [Etcher](https://etcher.io/)

Steps:
1. Open the tool
2. Select the Modicia O.S. ISO file
3. Choose your USB drive
4. Start writing

---

## 🚀 Step 3: Boot from USB

1. Insert the USB stick into your computer
2. Restart and enter BIOS/UEFI (usually by pressing `F2`, `F12`, or `DEL`)
3. Set the USB stick as the first boot device
4. Save and exit BIOS

Modicia O.S. will start in **Live Mode**, allowing you to try it without installing.

---

## 💾 Step 4: Install Modicia O.S.

Once in Live Mode:
1. Click on the **Install Modicia O.S.** icon on the desktop
2. Choose your language, keyboard layout, and time zone
3. Select installation type:
   - Install alongside existing OS
   - Erase disk and install Modicia O.S.
4. Follow the on-screen instructions to complete the installation

---

## ✅ Post-Installation Tips

- Update the system via the **Software Updater**
- Explore the **App Center** for additional software
- Customize your desktop with themes and widgets
- Enable the **Real-Time Kernel** if needed (one-click utility included)
